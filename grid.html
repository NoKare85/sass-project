Grid Project

This project demonstrates the use of CSS grid templates

You can change the size of the 'Projects' box by dragging from the bottom right corner or by clicking on the size icons for different display sizes on the top left corner.

Add a number of boxes you wish and see how the layout changes on different screen sizes. You can also click on a box to change it's width in the grid.
